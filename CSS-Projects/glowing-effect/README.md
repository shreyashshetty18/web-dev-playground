# Glowing Card
A small CSS-only card UI that features a rotating, glowing border.

## Features
- Animated conic-gradient border driven by [`@property --rotate`](CSS-Projects/glowing-effect/style.css).
- Simple responsive layout using the [`.box`](CSS-Projects/glowing-effect/style.css) class.
- No JavaScript required — pure HTML & CSS.

## Files
- [`index.html`](CSS-Projects/glowing-effect/index.html) — markup for the card.
- [`style.css`](CSS-Projects/glowing-effect/style.css) — styling and animation logic.
- [`cardimg.webp`](CSS-Projects/glowing-effect/cardimg.webp) — card image asset.

## Usage
1. Open [`index.html`](CSS-Projects/glowing-effect/index.html) in a browser.
2. Customize colors, border width, or animation duration in [`style.css`](CSS-Projects/glowing-effect/style.css).

## Notes
- Tweak the conic-gradient stops and `animation` timing to change the glow style and speed.
