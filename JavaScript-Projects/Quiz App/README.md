# Quiz App

A simple quiz application built with HTML, CSS, and JavaScript.

## Description

This is a multiple-choice quiz about JavaScript. The app presents a series of questions, and the user can select an answer for each. The app will then show the total score and the correct answers for all the questions.

## Features

- Multiple-choice questions
- Navigation between questions (previous/next)
- Score calculation
- Display of correct answers at the end

## How to Use

1. Open the `index.html` file in your web browser.
2. Answer the questions by selecting one of the options.
3. Use the "Previous" and "Next" buttons to navigate between questions.
4. After the last question, the app will display your total score and the correct answers.

## Technologies Used

- HTML
- CSS
- JavaScript
- jQuery
- Bootstrap
